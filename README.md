# Sapient tech test

Card validator is a simple card luhn validator 

## Installation

Unzip and cd into unzipped folder

To start the backend
```bash
cd backend
npm i && npm run start
```

To start the fronend
```bash
cd frontend
npm i && npm run start
```

Open http:localhost:3000/

Enjoy..
